# MiniProject-AI-Cybersecurity

**Project 16: Implementation of an AI-Based Cybersecurity Use Case**

## Overview
This project implements an AI-based Intrusion Detection System (IDS) using machine learning techniques to detect malicious network activities.

The system covers:
- Intrusion Detection (Supervised Learning)
- Anomaly Detection (Unsupervised Learning)
- Network Traffic Classification
- Adversarial Attacks Analysis
